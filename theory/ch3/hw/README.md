有两种不同的角度，

1. 从原动件出发，以 2 做参考系，

> 需要注意的是，考虑科氏加速度时，角速度应该是参考系的速度
>
> 而参考系是 2，又因为 2 和 3 的相对运动是滑动，所以 2 的角速度和 3 相等
>
> 所以用的是 3 的角速度

2. 从运动的习惯上思考，用 3 做参考系

> 这避免了参考系角速度关系不明确的的问题
>
> 而且，一般考虑相对运动也习惯从 3 理解到 2
